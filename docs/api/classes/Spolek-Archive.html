Spolek_Archive

Vytváří archiv uzavřeného hlasování (ZIP) + vede jednoduchý index na disku. Archivy se ukládají do wp-content/uploads/spolek-hlasovani-archiv/

Obsah ZIP:

  • summary.json
  • proposal.txt
  • minutes.pdf (pokud existuje)
  • votes.csv
  • audit.csv
  • mail_log.csv
